# AshLight Emoji Icon Studio

AshLight Emoji Icon Studio는 이모지 한 글자만으로 앱 아이콘과 스플래시 화면을 손쉽게 제작할 수 있는 정적 웹 도구입니다. iOS·Android·Web에서 요구하는 대표 해상도를 즉시 내려받을 수 있으며, 랜덤 프리셋과 커스터마이징 옵션을 함께 제공합니다.

## 주요 기능

- **이모지 기반 아이콘 생성**: 원하는 이모지와 그라데이션 색상을 지정하면 실시간으로 아이콘이 렌더링됩니다.
- **풍부한 스타일 컨트롤**: 그라데이션 각도, 모서리 라운드, 이모지 크기, 글로우 강도를 슬라이더로 조절할 수 있습니다.
- **자동 랜덤 프리셋**: `🎲 랜덤 조합` 버튼으로 색상·각도·크기가 다양한 조합을 즉시 적용할 수 있습니다.
- **다중 해상도 다운로드**: iOS(1024px), Android(512px), Favicon(192px)을 포함해 ZIP으로 묶어 한 번에 내려받을 수 있습니다.
- **스플래시 스튜디오**: 동일한 스타일로 1242×2688 해상도의 스플래시 이미지를 제작하고, 제목·보조 문구를 직접 입력하거나 랜덤 프리셋으로 설정할 수 있습니다.
- **실시간 미리보기**: 아이콘/스플래시 미리보기가 실제 렌더링 캔버스로 업데이트되어 다운로드 결과와 완전히 동일하게 표시됩니다.
- **후원 섹션 & 광고 스크립트 자리**: 서버 비용을 위한 후원 정보와 광고 스크립트를 추가할 수 있는 영역을 포함합니다.

## 로컬 실행 방법

이 프로젝트는 별도의 빌드 도구 없이 정적 HTML 하나로 구성되어 있습니다. 브라우저에서 `index.html`을 직접 열면 즉시 실행됩니다.

```bash
# macOS 예시 (현재 디렉터리가 프로젝트 루트라고 가정)
open index.html
```

혹은 간단한 정적 서버를 띄워 접근할 수도 있습니다.

```bash
# python3
python3 -m http.server 3000
# 그런 다음 브라우저에서 http://localhost:3000/index.html 접속
```

## GitHub Pages 배포 가이드

1. GitHub 저장소를 생성하고 `index.html`과 `README.md`를 업로드합니다.
2. 저장소의 **Settings → Pages**로 이동합니다.
3. Source를 `Deploy from a branch`로 설정하고, Branch는 `main`, 폴더는 `/root`를 선택합니다.
4. 저장 후 잠시 기다리면 `https://<사용자명>.github.io/<저장소명>/` 주소로 접근 가능합니다.
5. 커스텀 도메인을 사용하려면 Pages 설정에서 `Custom domain`을 입력하고 DNS에 CNAME 레코드를 추가합니다.

## 광고 및 분석 스크립트 추가

- `<head>` 영역에 Google AdSense 스크립트를 추가했습니다. 실제 환경에서는 발급받은 `client` 값을 적용해 주세요.
- 추가적인 통계(예: Google Analytics, Plausible 등)를 사용하려면 `<head>` 또는 `<body>` 하단에 스크립트를 삽입하면 됩니다.

## 주의 사항 & 팁

- 미리보기 이미지는 내보내기와 동일한 Canvas 기반 렌더링 결과를 사용하므로, 화면에서 보이는 것과 다운로드 결과가 완전히 일치합니다.
- 아이콘/스플래시 다운로드 후에는 앱스토어 혹은 웹 파비콘 규격에 맞춰 추가적으로 리사이징하거나 변환할 수 있습니다.
- 랜덤 프리셋은 스플래시 문구에도 적용되며, 필요 시 `randomSplashTexts` 배열을 수정해 고유한 카피를 추가하세요.
